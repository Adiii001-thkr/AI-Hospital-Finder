📌 Project Overview

AI Hospital Finder is a web application designed to help users quickly find nearby hospitals. The platform uses intelligent search and location-based features to display hospitals and healthcare facilities efficiently.

This project was built to demonstrate modern web development skills and create a helpful tool for users who need quick access to hospital information.

⸻

✨ Features
	•	🔍 Search for nearby hospitals
	•	📍 Location-based hospital suggestions
	•	⚡ Fast and responsive user interface
	•	💻 Fully responsive design for mobile and desktop
	•	🌐 Easy access to hospital information

⸻

🛠️ Technologies Used
	•	TypeScript
	•	React
	•	HTML5
	•	CSS3
	•	Vite
	•	Netlify (for deployment)
